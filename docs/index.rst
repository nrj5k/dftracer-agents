dftracer-agents
================

dftracer-agents packages the dftracer MCP server, the bundled agent skills,
and the pipeline subagents used to run annotation, tracing, diagnosis, and
optimization workflows.

This documentation describes the current repository behavior and the intended
usage patterns for each supported harness.
