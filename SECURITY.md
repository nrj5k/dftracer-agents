# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in dftracer-agents, please report it
privately rather than opening a public issue.

Contact: Hariharan Devarajan — hariharandev1@llnl.gov

Include as much detail as possible (affected version/commit, reproduction
steps, potential impact) so the issue can be triaged quickly. You should
expect an initial response within a few business days.
